# Cinema Booking System - Logbook

## Day 1
- Analyzed the project requirements and set up the initial folder structure.
- Created the basic index.html layout and defined the seat container.

## Day 2
- Developed the style.css to handle the visual states of seats (Available, Selected, Occupied).
- Implemented the layout for the cinema screen and seat rows.

## Day 3
- Started the JavaScript implementation.
- Created the Movie class in Movie.js to handle movie objects.
- Set up the js folder to modularize the code (Modularity).

## Day 4
- Built the api.js to fetch movie data from a local JSON server (db.json).
- Integrated the seat selection logic in app.js to allow toggling between 'Selected' and 'N/A'.
- Added dynamic calculation for the total price and seat count.

## Day 5
- Refined the code structure to be ESLint compliant.
- Added a fallback mechanism in the API for GitHub Pages compatibility.
- Final testing of the application and deployment to GitHub.
